## 二、技术选型

| 时间 | 方案 | 说明 |
|------|------|------|
| 2026-06-21 | 纯静态 HTML | 快速上线，直出 HTML 文件 |
| 2026-07-06 | Hugo | 引入静态站点生成器，模板复用、自动化构建 |

**Hugo 选型理由：**
- 模板一次改、全站生效（改导航/页脚不再手动改 149 个文件）
- Markdown 写内容，效率比手写 HTML 高 10 倍
- GitHub Actions 自动构建部署，git push = 发布

## 六、设计规范

- **字体栈**：`PingFang SC, Microsoft YaHei, -apple-system`（无 Google Fonts）
- **颜色**：CSS 变量体系（暗色 #0d0d1a，亮色 #f5f5f5，主色 #d4af37）
- **移动端**：max-width:480px 根字体 17px，触控区 44px+
- **图片**：WebP 格式，lazy loading，onerror 兜底
- **动画**：CSS transition/keyframes，grid-template-rows: 0fr→1fr 方案
