📖 历史与背景

2024年3月，微星（MSI）发布MSI Claw——这是首款搭载Intel Core Ultra处理器的PC掌机，也标志着主流PC硬件大厂正式进入掌机市场。

MSI Claw搭载了Intel Core Ultra 7 155H处理器（Meteor Lake架构），这是Intel专为轻薄本设计的处理器，集成Arc核显。7寸1920×1080分辨率、120Hz刷新率的IPS屏幕是当时PC掌机中规格最高的屏幕。

MSI Claw的发布意味着PC掌机市场不再只有Valve（Steam Deck）、华硕（ROG Ally）和联想（Legion Go）——传统PC硬件大厂的加入进一步验证了这个市场的潜力。不过Intel核显的性能和兼容性相比AMD方案仍有差距，MSI Claw的市场反响较为平淡。

🏆 市场表现与影响

  • 全球销量：约15万台（估算），低于ROG Ally和Legion Go
  • Intel vs AMD：MSI Claw是唯一采用Intel架构的主流PC掌机——Intel Arc核显的性能和驱动兼容性不如AMD RDNA架构
  • 屏幕优势：7寸1080p 120Hz屏幕是MSI Claw最大的卖点——分辨率和刷新率均优于Steam Deck OLED（800p 90Hz）
  • 后续型号：2024年下半年推出MSI Claw 8 AI+，升级为Intel Core Ultra 7 258V（Lunar Lake）处理器
  • 市场定位：面向追求高规格屏幕的PC掌机用户——但Intel处理器的性能和续航是主要短板

💡 趣闻与冷知识

  • MSI Claw是首款使用Intel Core Ultra处理器的掌机——此前所有主流PC掌机都使用AMD Ryzen APU
  • MSI Claw的120Hz屏幕在PC掌机中规格最高——但1080p分辨率对Intel Arc核显的压力较大，许多游戏需要降分辨率运行
  • MSI Claw的重量约675g——比Steam Deck（669g）略重，是主流PC掌机中较重的型号
  • MSI Claw的Thunderbolt 4接口可以外接eGPU——连接独立显卡后性能可接近桌面PC水平
  • MSI是继华硕和联想之后，第三家进入PC掌机市场的传统PC大厂——此后更多PC厂商可能跟进
