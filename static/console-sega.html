关于世嘉SEGA

1940年创立，曾是游戏产业的创新者。Mega Drive（Genesis）与SNES大战，索尼克成为吉祥物。Saturn策略失误让PS1崛起。Dreamcast是最后的主机，在线游戏先驱。2001年退出硬件市场，转型为第三方游戏开发商。代表作：索尼克、莎木、VR战士、街头赛车等。

家用主机

4款
1983
SG-1000
初代主机 · 与FC同年
世嘉首款家用主机，与FC同年发布但销量不佳。后续推出SG-1000 II改进型号。奠定了世嘉主机业务基础。
8位CPU 卡带介质 首款主机
1985
SEGA Mark III / Master System
欧美市场对抗FC
技术强于FC，但第三方支持不足。在巴西等市场成功，但在日本和美国被FC压制。代表作《Alex Kidd》。
8位增强 第三方不足 1300万台
1988
Mega Drive / Genesis
16位战争 · 索尼克诞生
对抗SNES，凭借《索尼克》系列赢得美国市场。Blast Processing营销策略。第三方支持改善，《怒之铁拳》《战斧》。
16位CPU 索尼克 3075万台
1994
SEGA Saturn
双CPU复杂架构 · 策略失误
首发在周三（错误决策），双CPU架构开发困难。EA放弃支持。虽然《VR战士》《莎木》优秀，但市场被PS1占领。
双CPU 开发困难 926万台
1998
Dreamcast
最后的世嘉主机 · 在线先驱
内置调制解调器，在线游戏先驱。Visual Memory Unit（VMU）副屏创新。《莎木》开创开放世界QTE。《Dreamcast》是世嘉最后的荣耀。
在线游戏 VMU副屏 913万台
